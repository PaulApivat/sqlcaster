About SQLcaster

SQLcaster's frontend is a static, client-side fetching page and its backend is an autogenerated REST API accepting SQL queries viaROAPIhosted onReplit. This site is intended to be fun, experimental way to explore messages (casts) on theFarcasterprotocol. Contributions and ideas would be wonderful - here is the open sourcerepo. SQLcaster was created by@whatrocks.
